# KAIROX IA

Sitio web institucional de **KAIROX IA** — Agencia de desarrollo web con Inteligencia Artificial.

## Stack

Sitio estático puro — sin frameworks ni build step.

- HTML5 semántico
- CSS3 (variables, grid, flexbox, animaciones)
- JavaScript vanilla (sin librerías externas)
- Google Fonts (Orbitron + DM Sans)
- Font Awesome (íconos)

## Estructura

```
.
├── index.html          Página principal
├── privacidad.html     Política de privacidad
└── Imagenes/           Recursos visuales
```

## Desarrollo local

Abrir `index.html` directamente en el navegador, o servir con cualquier servidor estático:

```bash
python -m http.server 3000
```

Luego visitar `http://localhost:3000`.

## Despliegue

Desplegado en Vercel desde la rama `main`.
